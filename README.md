# coding-project-template
#book review application with Node.js and Express
This server-side application, built with Node.js and Express.js, empowers users to store, retrieve, and manage book ratings and reviews. 
It offers functionalities such as retrieving a list of all books, searching for specific books by ISBN, author, and title, accessing reviews for specific books, user registration and login, adding/modifying/deleting book reviews (restricted to logged-in users), and concurrent multi-user access. 
Authentication is implemented using sessions and JWT.
